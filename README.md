## üì¶ Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/your-username/ems-suntec.git
   cd ems-suntec
   ```

2. **Install dependencies:**

   ```bash
   npm install
   # or
   yarn install
   ```

3. **Run the development server:**

   ```bash
   npm run dev
   # or
   yarn dev
   ```
   
Open http://localhost:3000 to view it in the browser.
